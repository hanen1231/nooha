# Nooha Catering Website Starter

موقع مبدئي ثابت لشركة إعاشة وتوريد غذائي، مبني بـ HTML وCSS وJavaScript بدون اعتماديات خارجية.

## هيكل الملفات

```text
.
├── index.html
├── about.html
├── services.html
├── projects.html
├── certificates.html
├── contact.html
└── assets
    ├── css
    │   └── style.css
    ├── js
    │   └── main.js
    ├── images
    │   └── .gitkeep
    └── documents
        └── .gitkeep
```

## ملاحظات التطوير

- الألوان والخطوط والقياسات الأساسية موجودة في `:root` داخل `assets/css/style.css`.
- الصور الحالية Placeholder من روابط خارجية ويمكن نقل الصور الحقيقية لاحقًا إلى `assets/images`.
- ملفات الشهادات والتراخيص المستقبلية يمكن وضعها داخل `assets/documents` وربطها في صفحة الشهادات.
- JavaScript الحالي يغطي القائمة المتجاوبة، تأثير الظهور، فلترة الأعمال، وسلوك نموذج التواصل المبدئي.
